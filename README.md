## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/gherghett/AdventOfCode2024.git
   ```
2. Navigate to the day’s folder:
   ```bash
   cd day04
   ```
3. Run the solution using your preferred C# compiler or IDE (e.g., `dotnet run`).
